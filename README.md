# Software Security Engineering

An Obsidian knowledge base for application security, built around the OWASP Top 10:2025.

Covers web vulnerability classes, identity and access, modern stacks (API, GraphQL, LLM), secure development, and cloud and supply-chain security. Each note follows the same structure: root cause, vulnerable code, how to find it, exploitation, and the fix.

Open the folder as an Obsidian vault to browse with linked navigation.
